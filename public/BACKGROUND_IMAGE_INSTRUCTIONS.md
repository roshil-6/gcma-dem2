# How to Add Background Image

## Steps:
1. Place your background image in the `public` folder
2. Name it exactly: `background.jpg` (or `.png`, `.jpeg`)
3. Supported formats: JPG, PNG, JPEG, WEBP

## Current Setup:
The Hero section will automatically use:
- `/background.jpg` (primary)
- Or you can tell me the exact filename and I'll update the code

## Image Requirements:
- Recommended size: 1920x1080 or larger
- Format: JPG, PNG, or JPEG
- The image will be automatically:
  - Set as full-screen background
  - Covered with a light dark overlay (40% opacity) for text readability
  - Responsive and scaled properly

## After Adding:
1. Save your image as `background.jpg` in the `public` folder
2. Refresh your browser (Ctrl+F5)
3. The background should appear behind the main text
